What the AI does with this knowledge

You (state: IDLE): "add reading, BP 132/84"
AI sees: no active session
→ "There's no active session right now. Do you want to start one first? I can open the pre-treatment form for you."
You (state: ACTIVE): "add reading, BP 132/84"
AI sees: active session, valid state for this command
→ prefill_reading(bp_sys:132, bp_dia:84) — opens form, waits for you to confirm
You (state: ACTIVE): "start a session"
AI sees: session already in progress
→ "You already have an active session (started today at 19:00). Did you want to add a reading or end the session?"
You (state: IDLE): "start session, weight 72.4, BP 140/85"
AI sees: valid, IDLE → PRE-TREATMENT
→ navigate_to('/treatment') + prefill_pre_treatment(...) — form opens filled, you tap Start
