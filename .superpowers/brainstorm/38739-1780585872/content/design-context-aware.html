What this enables

Context-smart tool calls
You: "BP 132/84, pulse 72"
AI knows you're in Add Reading, not Pre-treatment
→ prefill_reading(bp_sys:132, bp_dia:84, pulse:72)
blood_flow carries over from last reading automatically
Knows what's missing
You: "add reading"
AI sees bp_sys/dia/pulse are empty
→ "What's the BP and pulse for this reading?"
Doesn't guess, doesn't leave required fields blank
Navigates intelligently
You (on blood test screen): "I need to add a reading"
AI knows you're not on the treatment screen
→ navigate_to('/treatment') + opens Add Reading form
Cross-screen commands
You: "start session then show last Hb"
→ prefill_pre_treatment(...) [you confirm]
→ then navigate + filter_blood_tests
Sequences multiple tool calls in one message
