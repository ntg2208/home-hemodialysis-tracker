Chat Backend Architecture — Option C (Smart Summary)

Flutter chat UI → Cloud Run → Gemini AI Studio, with pre-assembled patient context

Request Flow

Flutter Chat Sheet
POST /api/chat { message, history }
Authorization: Bearer <MAIN_API_KEY>
↓
Cloud Run (Hono) — /api/chat
1. Verify Bearer auth
2. Assemble patient summary from live data
3. Build system prompt with summary + instructions
4. Forward [system, ...history, user message] to Gemini
Firestore
sessions + inventory
GCS
fitness summary
In-memory
blood tests JSON
↓
Gemini AI Studio
Model: gemini-2.5-flash (free tier)
Auth: API key from Secret Manager
Streaming SSE response back to Flutter
↓
Flutter Chat Sheet
SSE stream → typed words into markdown bubble
