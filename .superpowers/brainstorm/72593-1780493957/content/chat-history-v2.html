State Transitions

Chat opens → Active mode (new or resumed conversation, input enabled)
Tap 🕐 → History sheet (list of past conversations)
Tap a conversation → Read-only mode (loads messages, input disabled, "← Back" in header)
Tap "← Back" → History sheet again
Tap + → Active mode (new empty conversation, input enabled, title unset)
Tap ✕ → close sheet (active conversation auto-saved)

Auto-Save Behaviour

On each replyConversation saved to Firestore. If it's the first exchange, title is set from the user's first message.
On closeActive conversation saved. Empty conversations (no messages) are not persisted.
On "New chat"Current conversation saved first, then fresh state.
History orderingMost recent first (by updated_at).
