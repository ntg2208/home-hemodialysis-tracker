Design decisions
Library

@nivo/line replaces recharts. Removing recharts saves ~120KB; nivo/line adds ~180KB net.

Single line, 3 dot states
  • ● Out of range — always red, overrides timing colour
  • ● Pre-dialysis, in range — cyan
  • ● Post-dialysis, in range — amber
  • ● No timing, in range — indigo (matches line)
Reference range band

Rendered as a custom Nivo layer. Uses ref_low / ref_high from the most recent reading with both values present. Dashed indigo lines top and bottom, semi-transparent fill.

Latest value badge

Amber pill above the last data point. Colour matches the dot. Shows value + unit.

Tooltip

Shows date (e.g. "15 Jan 2024"), value + unit, timing (pre/post/—), and in/out of range status.

No zoom / brush

All data visible at once. X-axis auto-scales to date range. Labels formatted as "Jan '24".

Legend

Small inline dot legend below chart (pre / post / out of range), only shown when pre/post data exists for the marker.

Results table

Kept as-is below the chart. No changes needed.

Phase boundary lines

Two vertical reference lines (Oct 2023, Feb 2026) kept as custom layer markers.
