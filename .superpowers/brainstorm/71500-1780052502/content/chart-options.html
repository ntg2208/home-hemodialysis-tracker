Blood Test Chart — 3 Approaches

All three use the same Haemoglobin data (monthly, 2024–2026). Tap/click to select your preferred direction.

Haemoglobin (g/L)

A — Recharts (redesigned)

Same library, fixed config. Readable date labels, solid reference band, latest-value callout, red rings on out-of-range points. No new dependency.

Pros

  • Zero new bundle weight
  • Already integrated
  • Fast to ship

Cons

  • Still Recharts quirks on mobile
  • Limited annotation control
Haemoglobin (g/L)

B — Nivo (@nivo/line)

Smooth curves, filled area under line, pill-badge on latest value, filled red dots for out-of-range. Beautiful defaults tuned for dark themes.

Pros

  • Best defaults out of box
  • Built-in annotation layers
  • Great mobile tooltip

Cons

  • ~180KB new dependency
  • Different API to learn
Haemoglobin (g/L)

C — Apache ECharts

Full-featured charting engine. Built-in pinch-to-zoom and pan on mobile, markArea for reference bands, clean data zoom slider. Most powerful option.

Pros

  • Touch zoom/pan built in
  • Powerful reference band API
  • Very fast rendering

Cons

  • ~400KB (tree-shakeable to ~150KB)
  • Different paradigm (config object)
